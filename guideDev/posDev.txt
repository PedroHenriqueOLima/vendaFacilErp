GUIA DE DESENVOLVIMENTO DA TELA PDV

ELEMENTOS E AS POSSÍVEIS INTERAÇÕES DO USUÁRIO COM ELES:

NAVBAR:
    LOGO DO SISTEMA -> POSICIONAMENTO: À ESQUERDA SUPERIOR
    BOTÃO PARA ACESSAR O PAINEL DO ADMINISTRADOR -> POSICIONAMENTO: NO MEIO DA TELA -> É NECESSÁRIO INFORMAR A SENHA
    BOTÃO PARA FECHAR O SISTEMA -> POSICIONAMENTO: À DIREITA SUPERIOR
INPUT PARA INSERÇÃO E PESQUISA DE ITEMS:
    FUNCIONAMENTO -> O USUÁRIO PODERÁ PESQUISAR PELA DESCRIÇÃO DE O INICIO DA PESQUISA COMEÇAR POR UMA LETRA
    -> COMEÇANDO PELA LETRA -> SERÁ MOSTRADO UM MODAL COM A OPÇÃO DE FECHA-LO NO CANTO SUPERIOR DIREITO
    -> SERÃO MOSTRADOS PRODUTOS QUE CONTÉM AQUELA LETRA OU PALAVRA -> EM ORDEM ALFABÉTICA
    -> SERÁ MOSTRADA A DESCRIÇÃO, O VALOR E O ESTOQUE DAQUELE PRODUTO
    -> SE PASSAREM DE 10 ITEMS HAVERÁ UM SCROOL PARA DESCER E VISUALIZAR MAIS ITEMS
    -> A TELA ATRÁS DO MODAL FICARÁ DESFOCADA 
    -> AO TECLAR "ESC" OU CLICAR NO ICONE DE FECHAR O MODAL SERÁ FECHADO
    -> SE O USUÁRIO CLICAR EM CIMA DO ITEM COM UM DUPLO CLIQUE SERÁ INSERIDO O ITEM NA LISTA DE ITEMS DA VENDA
    -> O USUÁRIO TAMBÉM PODERÁ PESQUISAR PELO CÓDIGO DE BARRAS SE ELE DIGITAR UM NÚMERO NO INICIO, NÃO SERÁ ABERTO O MODAL
    -> O ITEM SERÁ MOSTRADO APENAS SE O CÓDIGO ESTIVER EXATO
    -> SE O USUÁRIO ESCANEAR COM LEITOR E O ITEM ESTIVER CADASTRADO ENTÃO SERÁ DIRETAMENTE INSERIDO UM ITEM
    -> EM TODAS ESSAS AÇÕES, SENÃO HOUVEREM REGISTROS ASSOCIADOS AO INPUT SERÁ MOSTRADO UM ALERTA AO USUÁRIO (NENHUM REGISTRO ENCONTRADO)
MENU DE BOTÕES: 
    POSICIONAMENTO -> LADO ESQUERDO
    BOTÕES: CONSULTAR ITEM, CADASTRAR PRODUTO, CADASTRAR SERVIÇO (APENAS PARA OFICINAS), CADASTRAR CLIENTE, CONTAS À RECEBER, LISTAGEM DE VENDAS
    BOTÃO "CONSULTAR ITEM" -> TECLA ASSOCIADA (F1) -> FUNCIONAMENTO: AO CLICAR OU TECLAR A TECLA ESPECIFICA, SERÁ ABERTO UM POP UP 
    -> NO POP-UP HAVERÁ UM INPUT COM A MESMA LÓGICA DO INPUT PARA INSERÇÃO E PESQUISA DE ITEMS
    -> UMA VEZ CONSULTADO, SERÁ/SERÃO MOSTRADOS REGISTROS, SE PASSAR DE 10 ITEMS TERÁ UM SCROOL
    -> O USUÁRIO TERÁ AS OPÇÕES DE ALTERAR O PREÇO, COM A SENHA DO ADMINISTRADOR
    -> OS DADOS DO ITEM QUE SERÃO MOSTRADOS SERÃO A DESCRIÇÃO, O ESTOQUE E O VALOR
    BOTÃO "CADASTRAR PRODUTO" -> SERÁ ABERTO UMA JANELA POP-UP COM UM FORMULÁRIO, CONTENDO OS CAMPOS -> CÓDIGO DE BARRAS, DESCRIÇÃO, CUSTO (PADRÃO: 0.00R$)
    -> PREÇO E E ESTOQUE ATUAL (PADRÃO 1)
    -> TODOS OS CAMPOS TEM QUE SER VALIDADOS, NÃO PODEM ESTAR VAZIOS -> APÓS INSERIR OS DADOS DO PRODUTO, APARECERÁ NA TELA O BOTÃO CADASTRAR SÓ SE OS DADOS 
    -> ESTIVEREM TOTALMENTE PREENCHIDOS -> SENDO CADASTRADO O PRODUTO O USUÁRIO VOLTARÁ PARA A TELA DE PDV
    BOTÃO "CADASTRAR SERVIÇO" OBS: (APENAS PARA OFICINAS) -> MESMA LÓGICA DE VALIDAÇÃO E APARECIMENTO DO BOTÃO -> APENAS OS DADOS SERÃO DIFERENTES (NOME E PREÇO)
    BOTÃO "CADASTRA CLIENTE" -> POP UP COM FORMULÁRIO PARA ESCOLHER TIPO DO CLIENTE -> FUNCIONAMENTO SE FÍSICO MOSTRAR OS CAMPOS DE NOME, CPF,TELEFONE RUA, BAIRRO, NÚMERO
    ->CIDADE (HAVERÁ UM INPUT COM MODAL DE LISTA DE CIDADES)
    -> SE JURIDICO -> NOME FANTASIA, RAZÃO SOCIAL, CNPJ, RUA, BAIRRO, NÚMERO, CIDADE, TELEFONE
    -> REGRA NÃO PODE SER CADASTRADO O CLIENTE SE O CPF OU CNPJ JÁ EXISTIREM NO BANCO DE DADOS
    -> BOTÃO "CONTAS À RECEBER" ->AO CLICAR ABRIRÁ UM POP-UP COM UM INPUT PARA PESQUISAR PELO NOME DO CLIENTE, AO DIGITAR POR EXEMPLO O NOME "PEDRO" APARECERÁ UMA LISTA DE TODOS OS CLIENTES
    QUE CONTENDO ESSE NOME -> AO CLICAR ENCIMA DO CLIENTE DESEJADO SERAM MOSTRADAS SUAS DIVIDAS A QUANTIDADE E O VALOR TOTAL DELAS E (UMA LISTA COM DATA DE COMPRA, DESCRIÇÃO E VALOR)
    -> EM CADA COMPRA HAVERÁ UM BOTÃO PARA VER OS DETALHES DA COMPRA (DATA DA COMPRA, PRODUTOS E OBSERVAÇÃO E VALOR) E UM BOTÃO DE QUITAR E A ESQUERDA UM BOTÃO DE EXCLUIR AQUELA CONTA  
    -> HAVERÁ TAMBÉM UM BOTÃO EMBAIXO DO POP-UP PARA QUITAR TUDO
    -> AO CLICAR EM QUITAR OU EM QUITAR TUDO APARECERÁ UM FORMULÁRIO PARA ESCOLHER A FORMA OU AS FORMAS DE PAGAMENTO, CARTÃO, PIX E DINHEIRO PODENDO FAZER COMPOSIÇÃO DE PAGAMENTO DINHEIRO + CARTÃO DINHEIRO + PIX CARTÃO + PIX
    -> AO CLICAR EM RECEBER QUE SERÁ O BOTÃO DO FORMULÁRIO SERÁ GERADO UM COMPROVANTE TENDO O NOME DO NEGOCIO O NOME DO CLIENTE O HORÁRIO DO PAGAMENTO E O VALOR DO PAGAMENTO DETALHADO AS FORMAS DE PAGAMENTO
    -> "BOTÃO LISTAGEM DE VENDAS" -> ABRIRÁ UM POP UP DE LISTAGEM DE VENDAS ENCIMA FICARÁ A FILTRAGEM DO PERIODO (DIA, MÊS, INTERVALO, ANO) A FILTRAGEM DO CLIENTE (ABRIRÁ A PESQUISA DE CLIENTES NA MESMA LÓGICA DA PESQUISA DE PRODUTOS PELA DESCRIÇÃO "PEDRO" LISTA TODOS OS PEDROS E POSSO ESCOLHER CLICANDO)
    -> HAVERÁ TAMBÉM UM BOTÃO DE RELATÓRIO (SERÁ EXIBIDO O FATURAMENTO DO DIA, DO PERIODO, DO ANO ETC, O VALOR TOTAL E O VALOR POR FORMA DE PAGAMENTO)
BOTÃO "CANCELAR VENDA":
    -> ACIONADO AO CLICAR OU CLICAR EM DELETE
    -> PEDIRA CONFIRMAÇÃO DE SENHA DO ADMINISTRADOR
    -> A TELA FICARÁ LIMPA E UMA NOVA VENDA PODERÁ SER EXECUTADA
LISTA DE PRODUTOS DA VENDA ATUAL:
    -> TABELA COM Nº DESCRIÇÃO, QUANTIDADE (PODERÁ SER ALTERADA NO INPUT QUE FICARÁ  DISPONIVEL PARA SER ALTERADO EM QUALQUER MOMENTO APENAS CLICANDO E EDITANDO)
    -> VALOR UNITÁRIO (PODERÁ SER ALTERADO CLICANDO ENCIMA, SERÁ EXIGIDA A SENHA DO ADMINISTRADOR E ALTERAÇÃO REFLETIRA NO BANCO DE DADOS) E O VALOR TOTAL DADO PELA QUANTIDADE * O VALOR UNITÁRIO
    -> OS ITENS FICARAM COM UM HOVER E PARA DELETAR DA VENDA BASTA CLICAR EM EXCLUIR (PEDIRÁ A SENHA DO ADMINISTRADOR)
CARDS (VALOR TOTAL E QUANTIDADE DE ITEMS)
    -> SERÃO ATUALIZADOS DE ACORDO COM  A LISTA DE ITEMS SE ACRESCENTADOS OU ALTERADOS OU EXCLUIDOS
BOTÃO PARA FINALIZAR A VENDA (CLICAR ENCIMA OU F12):
    -> SERÁ ABERTO UM POP-UP COM A OPÇÃO DE VOLTAR PARA A VENDA TECLANDO "ESC" 
    -> SERÁ EXIBIDO NOVAMENTE O VALOR TOTAL VENDA E A QUANTIDADE DE ITEMS 
    -> HAVERÁ UM FORMULÁRIO PARA AS FORMAS DE PAGAMENTO (DINHEIRO, CARTÃO, PIX, PROMISSORIA E CHEQUE)
    -> HAVERÁ AS OPÇÕES DE DESCONTO QUE PODEM SER PERSONALIZADAS HAVERÁ 2 OPÇÕES 5% OU 10% POR EXEMPLO
    -> APÓS DAR DESCONTO OU NÃO (O DESCONTO PADDRÃO É 0%) O USUÁRIO IRÁ PODER INFORMAR O CLIENTE OU DEIXAR SEM INFORMAR
    -> PARA INFORMAR TERÁ QUE CLICAR NO BOTÃO INFORMAR CLIENTE APARECERÁ UMA POP-UP COM UM  INPUT PARA PESQUISAR O CLIENTE SEGUINDO AS MESMAS LÓGICAS
    -> SE O CLIENTE FOR NOVO ELE PODERÁ CADASTRAR O CLIENTE NO BOTÃO CADASTRAR CLIENTE -> COM A MESMA LÓGICA DE CADASTRO DE CLIENTE, DEPOIS DE CADASTRAR O  CLIENTE CADASTRADO JÁ SERÁ ASSOCIADO AQUELA VENDA
    -> HAVERÁ O BOTÃO FINALIZAR A VENDA (PODE TAMBÉM CLICAR EM ENTER) HAVERÁ TAMBÉM O BOTÃO CANCELAR VENDA EM QUE SERÁ PEDIDA  A SENHA DE ADMINISTRADOR 
    _> E A VENDA SERÁ CANCELADA
